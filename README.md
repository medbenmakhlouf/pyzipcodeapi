Py Zip Code API
===============================

version number: 2.0.0

author: Mohamed Ben Makhlouf

python version: >= 3.10

Documentation
--------

link : http://pyzipcodeapi.readthedocs.io/en/v2.0.0/

Installation / Usage
--------------------

To install use pip:

    $ pip install pyzipcodeapi


Or clone the repo:

    $ git clone https://github.com/medbenmakhlouf/pyzipcodeapi.git
    $ python setup.py install
